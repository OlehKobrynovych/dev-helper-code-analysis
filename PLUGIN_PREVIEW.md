## Розміри

- **Ширина:** 600px
- **Висота:** 400-600px (залежно від результатів)
- **Popup:** Відкривається при кліку на іконку плагіна

## Кольорова схема

- **Заголовок:** Градієнт фіолетовий (#667eea → #764ba2)
- **Фон:** Світло-сірий (#f9fafb)
- **Акценти:** Фіолетовий, синій, зелений, помаранчевий
- **Текст:** Темно-сірий (#1f2937)

## Інтерактивні елементи

1. **Кнопка "Вибрати ZIP файл"**

   - При наведенні: підсвічується
   - При кліку: відкривається діалог вибору файлу

2. **Кнопки копіювання [📋]**

   - При кліку: копіює клас/функцію у буфер обміну
   - Змінюється на [✓] на 1 секунду

3. **Кнопки експорту**

   - JSON: завантажує детальний звіт
   - TXT: завантажує текстовий звіт

4. **Кнопка "Показати всі"**
   - З'являється якщо результатів більше 50
   - Розгортає повний список

## Особливості

- ✅ Адаптивний дизайн
- ✅ Плавні анімації
- ✅ Прокрутка для довгих списків
- ✅ Копіювання одним кліком
- ✅ Експорт у 2 форматах
- ✅ Детальна статистика
- ✅ Рекомендації по оптимізації

## Workflow

```
1. Користувач відкриває плагін
   ↓
2. Бачить кнопку "Вибрати ZIP файл"
   ↓
3. Вибирає архів проекту
   ↓
4. Бачить прогрес-бар "Аналіз проекту..."
   ↓
5. Отримує результати:
   - Статистика (4 картки)
   - Список невикористаних CSS класів
   - Список невикористаних функцій
   - Рекомендації
   ↓
6. Може:
   - Копіювати окремі класи/функції
   - Експортувати повний звіт (JSON/TXT)
   - Завантажити список класів окремо
```

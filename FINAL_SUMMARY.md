## 🚀 НАСТУПНІ КРОКИ:

### 1. Створіть іконки (30 секунд)

**Відкрийте у браузері:**

```
extension/create-icons.html
```

**Натисніть кнопку:**

```
"Створити та завантажити всі іконки"
```

**Результат:**

- Браузер завантажить icon16.png, icon48.png, icon128.png
- Помістіть їх у папку extension/icons/

### 2. Встановіть плагін (1 хвилина)

**Chrome / Edge / Brave:**

```
1. chrome://extensions/
2. Увімкнути "Режим розробника"
3. "Завантажити розпаковане розширення"
4. Вибрати папку: extension
```

**Firefox:**

```
1. about:debugging#/runtime/this-firefox
2. "Завантажити тимчасовий додаток"
3. Вибрати: extension/manifest.json
```

### 3. Використовуйте! (30 секунд)

```
1. Клік на іконку плагіна 🔍
2. Кнопка "Вибрати ZIP файл"
3. Завантажити архів проекту
4. Дочекатися аналізу (до 30 сек)
5. Переглянути результати
6. Експортувати звіт (JSON/TXT)
```
